系统要求

 

确保您的系统满足运行 Adobe Reader XI for Windows® 的最低要求：

 

・         1.3 千兆赫或更快的处理器

・         Microsoft® Windows XP with SP3（32 位）（或 SP2（64 位））；Windows Server® 2003 R2（32 位和 64 位）；Windows Server 2008 或 2008 R2（32 位和 64 位）；Windows 7（32 位和 64 位）；Windows 8（32 位和 64 位）

・         256 MB 已安装 RAM（推荐 512 MB）

・         300 MB 可用硬盘空间（注意：安装程序在安装过程中会生成临时文件，因此安装时所需的实际硬盘空间大于此值。）

・         1,024 x 768 屏幕分辨率

・         Internet Explorer 7、8、9、10；Firefox (ESR)；Chrome

 

对于 Windows Server 2003 SP2 和 Windows XP SP2（64 位)，需要安装 Microsoft 修补程序 KB-930627。请参阅：

http://support.microsoft.com/kb/930627
